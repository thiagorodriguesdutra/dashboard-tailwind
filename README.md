### Dashbord UI - Tailwind CSS

Este é projeto criado a partir de um template do Figma. 
Construído com Tailwind CSS totalmente responsivo.

Stack utilizada:

- Nextjs
- Reactjs
- Typescript
- Tailwind CSS
- Lucide Icons
- Figma
